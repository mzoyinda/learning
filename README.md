# PyClas-Backend
The backend for [our website](https://www.pyclas.com) .

## Installation

```bash
1. Clone the repo
2. npm install
3. npm start
4. Should be running on localhost:3000
```

## Endpoints 

1. /courseOfStudy?courseOfStudy=Chemical%20Engineering
2. /contactForm
